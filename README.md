# to-pggame-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
仓库内容主要为各类静态页面存档，不针对任何特定网站或服务。  
所有页面均为独立 HTML 文件，可直接在浏览器中打开查看。

## 仓库说明

本仓库主要用于存放以下类型的资源：

- 静态 HTML 页面归档
- 页面快照或离线版本
- 其他与页面相关的辅助文件

每个页面均以独立的 HTML 文件形式存在，不依赖外部服务器或数据库。

## 目录结构

```
to-pggame-notes/
├── README.md
├── pages/               # 存放所有 HTML 页面
│   ├── example.html
│   └── ...
├── assets/              # 页面引用的静态资源（如有）
│   └── ...
└── ...
```

- `pages/`：主要页面存放目录，每个 HTML 文件为一个独立页面。
- `assets/`：页面可能引用的图片、样式、脚本等资源文件（可选）。

## 页面归档说明

- 所有页面均为完整 HTML 文件，内含样式与脚本（如适用）。
- 页面内容保持原始状态，不进行修改或二次加工。
- 归档时间与来源信息可在页面内或提交记录中查看。
